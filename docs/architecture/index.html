⚡ 상태 흐름 다이어그램
사용자 액션이 API 호출을 거쳐 Context 상태를 갱신하고 컴포넌트에 반영되는 전체 흐름입니다. App.tsx가 소유하는 UI 상태(selectedNoteId, isCreating)와 Context가 관리하는 서버 상태(notes[])를 구분해 표현합니다.
